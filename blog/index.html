Let's learn more about HTML, CSS and JavaScript

HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

CSS

Cascading Style Sheets or CSS is a style sheet language used for describing the presentation of a document written in a markup language.

JS

JavaScript or JS is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.

HTML Tags

<a>: It is used to link from one page to another.

<body>: It contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

<br>: It inserts a single line break.

<button>: It defines a clickable button.

<div>: It defines a division or a section in an HTML document.

<em>: It is used to define emphasized text.

<footer>: It defines a footer for a document or section.

<form>: It is used to create an HTML form for user input.

<h1> and <h6>: They are used to define HTML headings. h1 is the biggest, h6 is the smallest heading.

<head>: It is a container for metadata (data about data) and is placed between the tag and the tag.

<header>: It represents a container for introductory content or a set of navigational links.

<hr>: It defines a thematic break in an HTML page.

<html>: It is the container for all other HTML elements (except for the tag).

<img>: It is used to embed an image in an HTML page.

<li>: It defines a list item.

<list>: It defines the relationship between the current document and an external resource.

<nav>: It defines a set of navigation links.

<ol>: It defines an ordered list. An ordered list can be numerical or alphabetical.

<p>: It defines a paragraph.

<span>: It is an inline container used to mark up a part of a text, or a part of a document.

<strong>: It is used to define text with strong importance. The content inside is typically displayed in bold.

<style>: It is used to define style information (CSS) for a document.

<title>: It defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

<ul>: It defines an unordered (bulleted) list.

<video>: It is used to embed video content in a document, such as a movie clip or other video streams..

CSS Properties

background-color: It specifies the background color of an element.

background-image: It specifies one or more background images for an element.

background-position: It specifies the position of a background image.

background-size: It specifies the size of the background images.

border: It sets the width, style, and color for all four sides of an element's border.

border-radius: It defines the shape of the border corners of an element.

color: It specifies the color of the text of an element.

display: It specifies how an element is displayed onscreen.

font-family: It defines a list of fonts for element.

font-size: It defines the font size for the text.

font-style: It defines the font style for the text.

font-weight: It specifies the font weight of the text.

height: It specifies the height of an element.

justify-content: It specifies how flex items are aligned along the main axis of the flex container after any flexible lengths and auto margins have been resolved.

list-style: It defines the display style for a list and list elements.

margin: It sets the margin on all four sides of the element.

margin-bottom: It sets the bottom margin of the element.

margin-left: It sets the left margin of the element.

margin-right: It sets the right margin of the element.

margin-top: It sets the top margin of the element.

padding: It sets the padding on all four sides of the element.

text-align: It sets the horizontal alignment of inline content.

width: It specifies the width of an element.

JavaScript Properties

getHours() returns the hour (0 to 23) of a date.

getMinutes() returns the minutes (0 to 59) of a date.

getSeconds() returns the seconds (0 to 59) of a date.

setDate() sets the day of the month of a date.

new Date() creates a new date object with the current date and time

getFullYear() returns the full year (4 digits) of a date.

getMonth() returns the month (0 to 11) of a date.

getDate() method returns the day of the month (1 to 31) of a date.

toDateString() method returns the date (not the time) of a date object as a string.

getDay() method returns the day of the week (0 to 6) of a date.

getElementById() method returns an element with a specified value

querySelector() method returns the first element that matches a CSS selector.

addEventListener() method of the EventTarget interface sets up a function that will be called whenever the specified event is delivered to the target.

setInterval() method calls a function at specified intervals (in milliseconds).

innerHTML property sets or returns the HTML content (inner HTML) of an element.

setMonth() method sets the month of a date object.
